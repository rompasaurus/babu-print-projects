# Stackable Resistor Storage Box

**Category:** Component-specific stackable drawer box
**Source:** [Printables — Various Size Stackable Resistor Storage Box](https://www.printables.com/model/7643-various-size-stackable-resistor-storage-box)
**Designer:** termlimit (remix of the original by [STTrife on Thingiverse](https://www.thingiverse.com/thing:3125601))
**License:** The Printables page does not state a license in a machine-readable way — check the page before re-uploading. Personal printing is fine.

## Why this one

This is the *classic* 3D-printed resistor drawer design. The boxes stack vertically (dovetail sides), slide out like drawers, each has a label strip, and termlimit's remix adds multiple box sizes so you can mix capacitor tubes and LED spools into the same stack. Battle-tested design from 2019 — tens of thousands of downloads, dialed-in tolerances for PLA.

## What to download

From the Printables page, grab:
- **Box** STL(s) — in whichever sizes you want (small / medium / large)
- **Drawer** STL matching each box size
- Optional **label template** (PDF) if it's attached

Save files into this folder.

## A1 print settings

| Setting | Value |
|---|---|
| Filament | PLA (keep friction low — PETG drawers stick in PLA boxes) |
| Layer height | 0.20 mm Standard |
| Walls | 2 |
| Infill | 10 % grid |
| Supports | None |
| Brim | None |
| Orientation | Open-face up for both box and drawer |

Print one box + one drawer first, verify the drawer slides smoothly, *then* batch-print 20 of them.

## Tips

- If drawers stick, sand the drawer rails lightly with 400-grit or bump XY compensation by +0.1 mm on the drawer print.
- Label the drawers with a Brother P-touch or just hand-write on masking tape.
