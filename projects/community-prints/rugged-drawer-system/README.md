# Rugged Drawer System — Gridfinity Stackable

**Category:** Sealed drawer cabinet (dust-proof) for Gridfinity bins
**Source:** [MakerWorld — Rugged Drawer System](https://makerworld.com/en/models/810461-rugged-drawer-system-gridfinity-stackable)
**License:** Check the page badge before redistributing. Personal printing is always fine.

## Why this one

Open Gridfinity bins collect dust and lint fast on a bench. This system puts a set of stackable, toolbox-style drawers *around* Gridfinity baseplates, giving you the same bin flexibility with closed drawer fronts and integrated labels. Looks like a shop-grade parts case, prints in PLA, and the drawers glide on printed-in rails without bearings.

## What to download

From the MakerWorld page grab:
- **Drawer chassis** (the outer box — usually sold in sizes like 1×6, 2×6 baseplate cells)
- **Drawer fronts** — matching drawer for each chassis slot
- Any **Gridfinity bins** you want inside the drawers (see [gridfinity-parametric-generator](../gridfinity-parametric-generator/))

Save files into this folder.

## A1 print settings

| Setting | Value |
|---|---|
| Filament | PLA (or PETG for drawer rails — they wear) |
| Layer height | 0.20 mm Standard; 0.16 mm Fine for the drawer fronts if you want crisp labels |
| Walls | 4 (rigidity — this is a structural part) |
| Infill | 20 % gyroid |
| Supports | Usually **None** — check each part; some label recesses may want paint-on support |
| Brim | 3 mm brim on the chassis bottoms — they have tall, narrow vertical walls |
| Orientation | As saved in 3MF |

Full 1×6 drawer stack is ~8 hours total print time.
