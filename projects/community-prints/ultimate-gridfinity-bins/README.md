## Why this one

The parametric generator is powerful but you still need to *decide* what shape fits your parts. This collection is a curated library of ready-to-print bins already tuned for common use cases — sorted small parts, screwdriver/plier racks, tall-component compartments, label-forward drawers. Grab the ones you want and skip the customizer.

## What to download

Browse the model's file list on the MakerWorld page and pick the bins you actually need. Good picks for electronics benches:

- **1×1 tiny-parts bin** — for SMD 0805 / 0603 components, one value per bin
- **2×1 multi-compartment** — jumper wires, heat-shrink
- **2×2 tall bin** — cap tubes, trimpots, headers
- **2×3 with divider** — perfboards, small PCBs standing on edge

Save files into this folder.
