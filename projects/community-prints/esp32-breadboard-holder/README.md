# ESP32 Dev Breadboard Holder

**Category:** Dev-board workstation — dual-breadboard base for wide-pinout MCUs
**Source:** [Printables — ESP32 Dev Breadboard Holder by Tom](https://www.printables.com/model/68303-esp32-dev-breadboard-holder)
**Designer:** Tom (@Tom_76704 on Printables)
**License:** Check the Printables page tag before redistributing. Personal printing is fine.

## Why this one

ESP32 dev boards are *wider* than a single half-size breadboard can straddle — one side of pins always ends up in the power rail, which kills one of the rails. This holder parks **two** breadboards with exactly the right gap between them so the ESP32 straddles the gap and every I/O pin gets its own breadboard column. Same trick works for Raspberry Pi Pico and any other wide-body board.

Pairs nicely with a [Multiboard wall tile](../multiboard-wall-system/) if you want to hang the whole prototyping rig on the wall when not in use.

## What to download

From the Printables page, grab the **base STL**. Some remixes add wire-management channels on the back — optional.

Note: the breadboards themselves are not printed — you still buy standard half-size breadboards (the kind with adhesive backing). Peel the backing off and stick them to the holder surface, or leave them loose.

## A1 print settings

| Setting | Value |
|---|---|
| Filament | PLA (any color — black or dark grey hides flux mess) |
| Layer height | 0.20 mm Standard |
| Walls | 3 |
| Infill | 15 % grid |
| Supports | None |
| Brim | None |
| Orientation | Flat-face down (as saved) |

~1.5 hours, ~25 g PLA.
