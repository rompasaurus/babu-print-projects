# Multiboard (Multibuild) Wall Storage System

**Category:** Modular wall-mounted storage system
**Source:** [multibuild.io](https://www.multibuild.io/) (formerly multiboard.io)
**License:** Multiboard/Multibuild core tiles and connectors are distributed free for non-commercial / personal use. Read the license on the site before redistributing — the project is community-driven and there are separate rules for remixes vs. re-uploads.

## Why this one

Multiboard is the "second-gen Gridfinity" — instead of just drawer bins, it's a full wall-mount tile system. Tiles lock together into a board of any size, and the same parametric hook interface used by Gridfinity-world accessories lets you hang bins, spool holders, cable clips, tool racks, dev-board mounts, anything. For a maker's electronics bench, this is how you turn a bare wall above the desk into *useful* storage without drilling pegboard.

## What to download

From [multibuild.io](https://www.multibuild.io/):
- **Core tiles** — the wall-mount base grid (start with a 3×4 tile run and expand)
- **Connectors** — stitch tiles together
- **Accessories** — bins, hooks, tool holders, spool holders

There are also MakerWorld mirrors of individual parts — search "Multiboard" on MakerWorld for one-click A1 prints if you prefer.

Save files into this folder.

## A1 print settings

| Setting | Value |
|---|---|
| Filament | PETG strongly recommended for the tiles (they carry load); PLA fine for accessories |
| Layer height | 0.20 mm Standard |
| Walls | 3 |
| Infill | 20 % gyroid for tiles; 15 % for accessories |
| Supports | None (system is designed support-free) |
| Brim | 3 mm brim on tiles |
| Orientation | As saved in the file |

One tile is ~45 min and ~20 g. A 4-tile × 4-tile starter wall is ~12 hours. Print one tile first, confirm it locks to its neighbor with the correct friction, then batch.

## Pairs well with

- Any electronics project that needs a fixed "home" on the wall — soldering iron holster, breadboard station mount, dev-board carrier.
