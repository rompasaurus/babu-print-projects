# Projects

Each subfolder is one printable thing. Convention:

- `README.md` — what it is, recommended print settings, photos later
- source CAD files (`.scad`, `.step`, `.FCStd`)
- exported `.stl` for direct slicing
- optional `.3mf` with known-good Bambu Studio settings baked in
