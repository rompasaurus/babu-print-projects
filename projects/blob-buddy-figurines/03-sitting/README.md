# Blob Buddy — Sitting

Wide flat base (body cut through its lower third), arms resting on
the ground in front. The widest, most stable of the three — great
first-print-of-the-family choice.

## Print

- **File:** [sitting.3mf](sitting.3mf)
- **Settings:** see [../README.md](../README.md). No brim needed — the
  flat base is ~26 mm diameter.
- **Time:** ~22 min on the A1 at 0.16 mm layers.

## Re-export from source

```bash
openscad -o sitting.stl sitting.scad
openscad -o sitting.3mf sitting.scad
```
