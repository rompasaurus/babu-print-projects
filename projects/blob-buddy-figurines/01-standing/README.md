# Blob Buddy — Standing

Classic pose: both arms at sides, two feet, looking forward.

## Print

- **File:** [standing.3mf](standing.3mf) — drop into Bambu Studio
- **Settings:** see [../README.md](../README.md). No supports, no brim.
- **Time:** ~25 min on the A1 at 0.16 mm layers.

## Re-export from source

```bash
openscad -o standing.stl standing.scad
openscad -o standing.3mf standing.scad
```
