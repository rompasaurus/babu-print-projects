# Blob Buddy — Waving

Right arm curves up and out, left arm at side.

## Print

- **File:** [waving.3mf](waving.3mf)
- **Settings:** see [../README.md](../README.md). **Add a 3 mm brim**
  — the raised arm makes the silhouette taller and narrower, so the
  brim keeps it planted.
- **Time:** ~30 min on the A1 at 0.16 mm layers.

## Re-export from source

```bash
openscad -o waving.stl waving.scad
openscad -o waving.3mf waving.scad
```
