# Kirby — Sitting (Easy to Print)

**Source:** [Kirby - Easy to Print by ChaosCoreTech on Thingiverse](https://www.thingiverse.com/thing:1913526)

One of the most popular Kirby prints ever. Sitting pose with arms on
the ground; the bottom is sliced flat to guarantee bed contact. No
supports needed.

## Download

1. Open the Thingiverse link (no account needed).
2. Click **Download All** or pick the specific `.stl`.
3. Save into this folder as `kirby-sitting.stl`.

## A1 print settings

- Scale to **50 mm tall** (ChaosCoreTech's original is ~60 mm; scale down).
- Layer height **0.16 mm**.
- Pink PLA Basic.
- No supports, no brim required — the flat base is big.
- Seam: "Aligned, back" keeps the seam off the face.
