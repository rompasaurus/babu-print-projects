# Kirby — Color-Sorted Print (No AMS Required)

**Source:** [Kirby - Color-Sorted Print by Mmm on MakerWorld](https://makerworld.com/en/models/705934-kirby-color-sorted-print-no-ams-required)

Single-model Kirby designed so **all same-color regions are at the
same height**. You print it and do manual filament swaps at layer
transitions via Bambu Studio's "Change Filament at Layer" feature.
Gets you multi-color without an AMS.

## Download

1. Open the MakerWorld link.
2. Download the `.3mf`.
3. Save into this folder as `kirby-color-sorted.3mf`.

## A1 print settings

**Heads up:** the MakerWorld page notes that **supports are required
for the main body** on this one (unlike #1 and #3). Enable them in
Bambu Studio.

- Scale to **50 mm tall**.
- Layer height: **0.20 mm** — matches the color-region boundaries
  baked into the model. Don't override.
- Supports: **Yes, normal** (painted supports are cleaner but normal
  works).
- Filament swaps: Bambu Studio will ask you to pause at the correct
  layers. Follow the prompts.

Plan the swap sequence before starting:

1. **Red** (feet) — layer 1 to ~N1
2. **Pink** (body) — N1 to N2
3. **White** (eye whites) — N2 to N3
4. **Black** (pupils, mouth) — N3 to N4
5. **Light pink** (cheeks) — final layers

Exact layer numbers come from the `.3mf` in Bambu Studio's preview.
