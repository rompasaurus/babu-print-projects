# Kirby — Classic Standing (No Supports)

**Source:** [Kirby - No supports by BODY3D on Printables](https://www.printables.com/model/178753-kirby-no-supports)

The default "hello, I am Kirby" pose. Designed from the ground up to
print without supports — a flat chunk is taken off the bottom of the
feet so the model sits on the bed with full base contact.

## Download

1. Open the Printables link above.
2. Download the `.stl` (or `.3mf` if offered).
3. Save into **this folder** as `kirby-standing.stl`.

## A1 print settings

Use the defaults in [../README.md](../README.md). Specifically:

- Scale to **50 mm tall**
- No supports, no brim needed
- Pink PLA Basic
- ~45 min print time at 0.16 mm layers

## Add a local .gitignore

Since Kirby STLs shouldn't be committed to the public repo, drop a
`.gitignore` in this folder (one-liner):

```bash
echo '*.stl' > .gitignore
echo '*.3mf' >> .gitignore
```
