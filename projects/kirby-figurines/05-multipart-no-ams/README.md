# Kirby — Multi-part, No AMS

**Source:** [Kirby multi-parts NO AMS by x4mp0 on MakerWorld](https://makerworld.com/en/models/524217-kirby-multi-parts-no-ams)

Kirby split into multiple STL pieces (body, feet, cheeks, eyes) so you
can print each in its own color on a **single-nozzle** printer without
needing the AMS lite. Assembles with a drop of glue or a friction fit.

## Download

1. Open the MakerWorld link.
2. Download the bundle (usually a `.zip` containing multiple `.stl`
   files or a multi-part `.3mf`).
3. Extract into this folder. You should end up with something like:
   ```
   05-multipart-no-ams/
     body.stl
     feet.stl
     cheeks.stl
     eyes.stl
     ...
   ```

## A1 print settings

Print each piece on its own plate, in the matching filament.

| Part | Filament | Notes |
|---|---|---|
| Body | Pink PLA | Largest piece; ~30 min at 50 mm scale |
| Feet | Red / Dark Pink PLA | 2 small pieces; print both together |
| Cheeks | Light Pink PLA | Tiny, consider brim to keep them stuck |
| Eyes / mouth | Black PLA | Tiny; sharp detail needs 0.12 mm layers |

- Scale all parts by the **same factor** — 0.7x for a 50 mm Kirby.
  If one part gets scaled differently, it won't fit.
- Glue with CA (superglue) or plastic-weldable PLA cement.
