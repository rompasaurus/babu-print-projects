# Kirby — Sleepy / Sleeping

**Source:** [Sleepy KIRBY on Cults3D](https://cults3d.com/en/3d-model/game/sleepy-kirby)

Kirby curled up, eyes closed. Flat-ish bottom profile makes it easy to
print and fun to put on a nightstand or monitor stand.

## Download

1. Open the Cults3D link. You may need a free account to download.
2. Pick the `.stl`.
3. Save into this folder as `kirby-sleepy.stl`.

## A1 print settings

- Scale to **40–45 mm** long axis — sleepy Kirby looks best slightly
  smaller than the standing version.
- Pink PLA Basic.
- **Check orientation in Bambu Studio.** If the bottom isn't flat,
  rotate so the belly rests on the bed.
- Layer height 0.12 mm for the closed-eye detail to show well.
- Supports: usually not needed for a curled pose; Bambu Studio will
  warn you if there's a steep overhang.
